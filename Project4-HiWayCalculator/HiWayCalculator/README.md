# Hi-Way Calculator

HD Hyundai 근무 시간 계산기 - Hi-HR 데이터 기반

## 사용 방법

`HiWayCalculator.html` 파일을 더블클릭하여 실행하세요.

## 기능
- ✅ Hi-HR 데이터 자동 파싱
- ✅ 시수(Sisu) 계산 (소수점 이월 처리)
- ✅ 동적 대시보드 (진행률, 누적시간)
- ✅ 월별 캘린더 뷰 (색상 그라데이션: 노랑→초록→빨강)
- ✅ 초과 달성 시 네온 펄스 효과
- ✅ 단일 HTML 파일 (공유 용이)

## 시스템 요구사항
- 최신 웹 브라우저 (Chrome, Edge, Firefox 등)
